How to edit this page

Change the roster in one place

  • Open assets/data.js and scroll to the players array.
  • Edit names, nicknames, positions, bios, and strengths there. Use the Game Logs page if you want the stats to update game by game.
  • Keep each id unique so the player cards still work.
  • Refresh the browser and this page rebuilds itself automatically from the current season data.

  • Give each player a season timeline or individual tape playlist.
